# asteroids_python
asteroids game with python and OOP, all credits to realpython turoial 


## How to run

create a virtual environment
```
python -m venv .env
```

activate virtual environment
``` 
source .env/bin/activate
```

Install dependencies
```
 pip install -r requirements.txt  
```

Run the game
```
python space_rocks   
```
